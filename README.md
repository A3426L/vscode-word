# CSV暗記カード (Flashcards for VS Code)

VS Codeのエクスプローラ（サイドバー）でCSVファイルを読み込んで、手軽に単語帳や暗記カードとして学習できる拡張機能です。

## 🌟 主な機能

- **エクスプローラ統合**: サイドバー（`Flashcards`セクション）からいつでも学習を開始できます。
- **3Dカードアニメーション**: カードをクリックすると裏返り、答えを確認できます。
- **インテリジェント・ブックマーク (🔖)**: 
  - 現在の位置（何枚目か）を記憶し、次回もその場所から再開できます。
  - **シャッフル状態も保持**: シャッフルした後のランダムな並び順もそのまま記憶して復元します。
- **多機能シャッフル (🔀)**:
  - 1クリックでカードをランダムに並び替えます。
  - 再度クリックすると、**元のCSVの順番**に一発でリセットできます。
- **CSV自動ロード**: VS Codeを起動した際、前回最後に開いていたCSVファイルを自動的に読み込みます。
- **スキップ機能**: CSVの1行目（見出し行）は自動的に読み飛ばされるため、データを編集せずそのまま読み込めます。

## 🚀 使い方

1.  **インストール**: 拡張機能をインストールすると、サイドバーに「Flashcards」のアイコン（またはセクション）が表示されます。
2.  **CSVのロード**: タイトルバーの **[＋]** アイコン、または「Select CSV」ボタンからCSVファイルを選択します。
3.  **学習開始**:
    - `Prev` / `Next` ボタンでカードを移動します。
    - カードをクリックして裏返します。
4.  **状態の保存**: 中断したいときは **[🔖]** アイコンをクリックしてブックマークしてください。

## 📁 推奨されるCSV形式

CSVファイルは、**「1枚目が問題、2枚目が答え」**のカンマ区切り形式で作成してください：

```csv
見出し1, 見出し2 (ここは自動で無視されます)
Apple, りんご
Banana, バナナ
Cherry, さくらんぼ
```

- 文字コードは `UTF-8` を推奨します。
- 1行目は自動的に無視されるため、自由に項目名を入力して大丈夫です。

## 🛠️ タイトルバーのアイコン
- **[＋]**: 新しいCSVファイルを読み込む
- **[🔀]**: シャッフル / 元の順番に戻す（トグル）
- **[🔖]**: 現在の場所と並び順を記憶する（ブックマーク）
